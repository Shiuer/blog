原来直接加个chrome私有属性就能解决的问题不知道从哪版chrome升级后不行了

webkit的私有属性：

html{-webkit-text-size-adjust:none;}

所以只能找别的解决办法 看别人的解决方案是利用CSS3的 transform属性 transform:scale()

-webkit-transform:scale();必须用在块级元素 否则不起效果 而且缩放只能是缩放文字 占的位置仍然是原来字号大小的位置 所以要给margin负边距 margin:-5px -10px; 然 新版的 opera 也用 webkit内核 所以要给它缩回去 只有加私有属性了 -o-transform

综上得出

{-webkit-transform:scale(0.94);-o-transform:scale(1);margin:15px -10px 0 -20px;display:block;}
